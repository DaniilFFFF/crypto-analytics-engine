# Crypto Portfolio Analytics Engine (ELT Pipeline)

## 📌 Про проект
Це комплексний проект з аналітики криптовалютного портфеля, побудований за принципом **ELT (Extract, Load, Transform)**. Проект автоматизує збір ринкових даних через API та розраховує ключові фінансові метрики (PnL, ROI, кумулятивний баланс) за допомогою Python та SQL.

## 🛠 Технологічний стек
- **Мова:** Python (Pandas, Requests)
- **API:** CoinGecko (історичні та актуальні ціни)
- **Database/SQL:** SQLite / PandasSQL
- **BI:** Power BI

## 🏗 Структура проекту
- `/data`: Сирі дані транзакцій (CSV).
- `/scripts`: Python-скрипти для ETL процесів.
- `/sql`: Аналітичні запити та трансформації.
- `/notebooks`: Прототипування та візуалізація.

## 📈 Ключові метрики
- **Cost Basis:** Середньозважена ціна закупівлі активів.
- **Unrealized PnL:** Поточний прибуток/збиток на основі живих ринкових цін.
- **Running Total:** Динаміка зміни капіталу в часі.
